## Documentation Sections

The documentation is organized for easy access by different roles:

* **For Users & Admins**: Functional guides on orders, financial operations, and system management.
* **For Web Developers**: Technical deep-dives into the Laravel 12 API and Vue 3 dashboard.
* **For Mobile Developers**: Full guide to the Flutter app using the Cubit state management pattern.
* **For System Admins**: Maintenance, deployment, and troubleshooting guides.
